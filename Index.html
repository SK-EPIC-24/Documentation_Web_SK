Introduction

JavaScript is a versatile, high-level programming language used primarily for web development.
It allows developers to create dynamic and interactive web pages. JavaScript runs in the browser,
enabling client-side scripting and server-side functionalities through environments like Node.js.

What You Should Already Know

Before learning JavaScript, it is helpful to have a basic understanding of:
HTML: Structure of web pages.
CSS: Styling and layout of web pages.
Programming Concepts: Variables, functions, loops, and conditions.
Familiarity with these topics will help you grasp JavaScript concepts more efficiently.

JavaScript and Java

Despite their similar names, JavaScript and Java are distinct programming languages.
Java is a statically typed, object-oriented language that requires compilation.
JavaScript is a dynamically typed, interpreted language used mainly for web development.
JavaScript is lightweight and commonly embedded in HTML, while Java runs on a separate virtual machine (JVM).

Variables

In JavaScript, variables are used to store data values. JavaScript provides three ways to declare variables:
Using var (Old method, function-scoped)
var message = "Hello, JavaScript!"; console.log(message);
Using let (Block-scoped, preferred for mutable variables)
let name = "Alice"; name = "Bob"; // Allowed console.log(name);
Using const (Block-scoped, for constants)
const pi = 3.14159; console.log(pi); // pi = 3.14; // Error: Assignment to a constant variable.
Understanding these fundamental concepts will help you build more complex JavaScript applications.
